### Overview

This IBM Redpaper describes Row and Column Access Control support in IBM DB2 for i database system. The document covers implementing security controls at the row and column level within the DB2 database.

Key concepts covered include:
- Role-based access control (RBAC) implementation
- Separation of duties in database administration
- Row-level security permissions
- Column-level masking for sensitive data protection

The Redpaper provides technical guidance for administrators and architects implementing data security in IBM DB2 for i environments.
